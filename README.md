# Better Swords

A vanilla-friendly 16x Minecraft resource pack.

## Features

- Improved sword textures
- Clean vanilla style
- Lightweight
- Easy to use

## Installation

1. Download the resource pack.
2. Place it inside the resourcepacks folder.
3. Enable it from Minecraft.

## Versions

Minecraft 1.21+
